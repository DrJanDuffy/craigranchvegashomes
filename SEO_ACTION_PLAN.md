## üö® **TROUBLESHOOTING**

### If Pages Aren't Indexing:
1. Check robots.txt isn't blocking
2. Verify no "noindex" tags
3. Check for crawl errors in Search Console
4. Request indexing manually
5. Wait 1-2 weeks for Google to crawl

### If Structured Data Has Errors:
1. Use Schema Validator to identify issues
2. Check JSON-LD syntax
3. Verify required properties are present
4. Fix errors and resubmit
5. Wait 1-2 weeks for re-crawl

### If Knowledge Panel Not Appearing:
1. Ensure Organization schema is correct
2. Verify logo is accessible
3. Check business information is complete
4. Wait 2-4 weeks (can take time)
5. Use Knowledge Panel feedback if incorrect
